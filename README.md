# 🐄 CowCare: PMK Monitor
**Sistem Deteksi Dini dan Monitoring Penyakit Mulut dan Kaki (PMK) pada Hewan Ternak**

[![Tech: React](https://img.shields.io/badge/Tech-React-61DAFB?logo=react&logoColor=white)](#)
[![Stage: Capstone Project](https://img.shields.io/badge/Stage-Capstone-orange)](#)

CowCare adalah solusi digital yang dirancang untuk membantu peternak melakukan deteksi dini terhadap gejala PMK melalui kuesioner diagnostik mandiri. Dengan parameter suhu tubuh dan observasi fisik (mulut, hidung, kaki), sistem memberikan persentase kemungkinan indikasi PMK dan solusi penanganan yang tepat.

## 🚀 Fitur Utama
- **Diagnostik Mandiri:** Kuesioner 10 tahap berdasarkan gejala klinis PMK.
- **Visual-Based Choice:** Pilihan jawaban dilengkapi referensi gambar untuk akurasi input user.
- **Persentase Indikasi:** Kalkulasi otomatis untuk menentukan tingkat risiko ternak.
- **Pusat Informasi:** Edukasi mengenai pencegahan dan penanganan pertama pada PMK.
